Ý tưởng : lấy ý tưởng từ game nyan cat fly

Em chọn game này vì nhìn game hay và vui nhộn

cách chạy chương trình: chạy và bấm start

các chức năng chính:
-pause:
+continue: tiếp tục game
+restart: chơi lại
+menu: trở về menu

-game over:
+restart: chơi lại game
+menu: trở về menu;

đề xuất hướng cải thiện tiếp theo:
+có phân loại độ khó
+có các vật phẩm đặc biệt
+có cài đặt
